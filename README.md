# KVM-console 是一款轻量的KVM管理工具

 适合单台服务器创建管理虚拟机管理，快速搭建测试环境
 
 服务器操作系统：Ubuntu Server 20.04
 数据库：Sqlite
 VNC：novnc
 
 
 安装方法：解压根目录下kvm.tar.gz至/opt/目录中，执行/opt/kvm/install.script脚本 
 连网状态下自动完成需要的组件安装
